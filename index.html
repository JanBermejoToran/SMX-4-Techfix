Problemas de Hardware

- El ordenador (torre) no detecta el disco duro


- El primer paso que debes realizar es reiniciar el ordenador... Si esto no funciona debemos comprobar que esten bien conectados los cables que van del disco duro a la placa base. Para hacerlo mas facil los desconectaremos y los volveremos a conectar. Si no funciona intenta a sustituir el cable que hay por uno nuevo. Si esto sigue sin funcionar debes robar el paso dos.
- Para realizar el paso dos tenemos que: Administrador de dispositivos > Unidades de disco > Haz clic derecho en tu disco duro externo > Actualizar controlador > Buscar controladores automáticamente

-LEl ordenador (torre) no se enciende y suenan cuatro BEEP


Opción 1: Si suenan 4 Beeps significa que es error en la memoria RAM

- Para solucionar este error tenemos que comprobar que la RAM este bién colocada o que no este rota, si esta rota sustituyela por una nueva y prueba nuevamente.

La targeta RAM es la encargada de subir la informacion desde el disco duro a la CPU por eso no se enciende.

El ordenador (torre) no se enciende y no suena ningun BEEP


Este problema se debe a que la CPU esta dañada. Revisa que los pines del socket no esten doblados y si hay algun pin doblado hay que llevarlo a reparar y posiblemente inteneten enderezar un pin y si no lo consiguen tendran que cambiar la placa base.

No me funciona el USB del panel frontal

Tienes que revisar que el cable que va desde el panel frontal a la placa base este bien enchufado o enchufado en el lugar correcto.
